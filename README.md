# 프로젝트 소개
- 세컨드 핸드 프로젝트에 뜬금 없이 추가된 위치기반 식당추천 시스템입니다


#  기능 설명
1. 위치 정보를 기반으로 카카오 맵 api를 이용해 내 위치를 띄웁니다.
2. 주변 식당을 찾습니다.
3. 내가 원하는 식당 종류(한식 양식 일식 등)을 선택하면 해당 식당의 정보가 화면에 출력됩니다.
